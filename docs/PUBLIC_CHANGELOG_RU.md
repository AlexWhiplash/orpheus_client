# Публичный changelog (RU) — для админки Orpheus

Это **пользовательский** список изменений на русском для экрана «История обновлений»
в приложении. Клиент берёт его с сервера: `GET /api/public/releases`
(поля `version_name`, `created_at`, `public_changelog`).

**Что сделать:** для каждой версии ниже создать/обновить запись в админке
(раздел «Versions») и вставить текст блока в поле `public_changelog`.
Порядок — новее сверху. Даты выставить по факту публикации.

> Проверка после заливки:
> `curl https://api.orpheus.click/api/public/releases?limit=50`
> должен вернуть эти записи (сейчас отдаёт пусто → в приложении виден офлайн-фолбэк).

## version_name: `1.1.7`  (LATEST)

```
- Удаление комнаты у всех: владелец теперь может полностью удалить свою комнату со всеми сообщениями у всех участников (раньше можно было только выйти или очистить историю).
- Обращение в поддержку до активации: кнопка «Поддержка» доступна прямо на экране ввода кода — можно написать разработчику, даже если лицензия ещё не активировалась.
- Поиск по контактам: быстрый фильтр по имени в списке контактов.
- Подсказка при первом запуске: короткое объяснение, как поделиться своим ID и добавить собеседника.
- Шифрование данных на устройстве: вся база (переписка, контакты, заметки) теперь зашифрована на диске — данные защищены при потере или изъятии телефона.
- Доставка звонков и сообщений без сервисов Google: приложение самостоятельно поддерживает связь, когда свёрнуто или закрыто.
- Стабильность звонков: исходящий звонок больше не звонит бесконечно при отсутствии ответа; ответ на звонок с заблокированного экрана срабатывает надёжнее.
- Исправления по итогам тестирования: корректный вход после активации лицензии, устранены зависания загрузки и таймера блокировки, убран лишний значок непрочитанного на иконке, поправлено отображение русского интерфейса.
```

## version_name: `1.1.4`

```
- Усилены настройки сети и безопасности.
- Экспорт аккаунта: если биометрия недоступна, подтверждение по PIN приложения.
- Массовое выделение сообщений для удаления (долгое нажатие или меню).
- Компактные отметки о звонках в переписке; одиночное нажатие по ним больше не набирает номер повторно.
- Данные о регионе хранятся только на устройстве и не передаются на сервер.
- Улучшена локализация интерфейса (русский и английский).
```

## version_name: `1.1.0`

```
- PIN-код (6 цифр) — дополнительная защита входа.
- Код принуждения (duress): второй PIN, который показывает пустой профиль.
- Код стирания с подтверждением удержанием — защита от случайного удаления.
- Автостирание после нескольких неверных попыток (опционально).
- Экран «Как пользоваться» — короткое руководство по функциям и рискам.
- Стабилизация звонков: заблокированный экран больше не мешает отвечать и разговаривать.
```

## version_name: `1.0.0`

```
- Полноценные звонки в спящем режиме: входящие проходят, даже если телефон заблокирован или приложение закрыто.
- Новый экран «Монитор системы»: графики стабильности сети, пинг и статус шифрования в реальном времени.
- Обновлённая навигация: Контакты, Система, Профиль.
- Исправлено соединение при ответе на звонок из уведомления.
- Обновлён дизайн профиля и QR-карточки; собственная иконка и экран загрузки.
- Поле ввода поддерживает многострочный текст; анимация шифрования при отправке.
- Устранено дублирование звонков, повышена стабильность соединения.
```

_Примечание: оплата лицензии в приложении намеренно не включена в список — серверная
