# Orpheus — задание на тестирование (тестовая сборка)

**Сборка:** v1.1.6 (build 12), тестовая
**Платформа:** Android (реальное устройство)
**Файл:** `orpheus-v1.1.6-b12-test-screenshots.apk` (прилагается)

> ⚠️ **Это спец. тестовая сборка:** в ней **разрешены скриншоты** (в обычной версии они заблокированы ради безопасности). Это временно, только для теста.

## Что нужно
- **Один Android-телефон.**
- **Ключ активации (лицензия)** — запросить у заказчика.
- **Скриншоты работают** — делайте их к каждому найденному багу.

## Как ставить
1. Удалить старый Orpheus, если стоял.
2. Установить APK (разрешить установку из неизвестных источников).
3. Открыть → онбординг → ключ активации → создать аккаунт.

## Как отчитываться
По каждому пункту: **ОК** / **БАГ**. Для бага:
- Шаги (что делали), что ожидали, что вышло.
- **Скриншот** (теперь работают!).
- Модель телефона + версия Android.
